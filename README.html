Overview

What is Campus Cash Pro?

Campus Cash Pro is an all-in-one personal finance toolkit designed specifically for college students. It lives entirely in a single HTML file — no installation, no subscription, no account required.

Everything in one file

Track your monthly budget, log transactions, manage sinking funds, set savings goals, monitor subscriptions, track debt payoff, log GPA, record scholarships and internships, and keep a money journal — all from a single file that opens right in your browser.

Your data stays on your device. Nothing is ever sent to a server.

Getting Started

How to open Campus Cash Pro

No installation needed. Just open the file.

1
Locate the file
Find CampusCashPro.html in the folder where you downloaded it. Keep this folder somewhere safe — like your Documents folder or Desktop.
2
Open in your browser
Double-click the file to open it in your default browser. It works in Chrome, Safari, Firefox, and Edge. No internet connection required after the first load.
3
Complete the quick setup
The first time you open it, a short welcome screen will guide you through entering your name, school, and choosing a color theme. Takes about 60 seconds.
4
Bookmark it (optional)
Drag the file onto your browser's toolbar or bookmark bar for quick one-click access anytime.
Your Data

Where is your data stored?

All your financial data is stored locally in your browser using a technology called localStorage. No server. No cloud. No account.

Everything you enter — transactions, budget amounts, goals, notes — is saved automatically to your browser's local storage on your device. It works like a file on your computer: only you can see it, and it only exists on the device and browser you used to create it.

Important: Your data is tied to the specific browser and device you use. If you open the file in a different browser or on a different device, you will start fresh. Use the export feature in Settings to back up and transfer your data.
Do not use Private / Incognito mode as your regular window. Private browsing clears all local storage when the window is closed, which means your data will be permanently lost at the end of each session. Always use a normal browser window.
Backups

Exporting and importing your data

Campus Cash Pro has a built-in backup system so you never lose your data.

How to export a backup

Go to Settings (the gear icon in the sidebar) and click Export Data. This downloads a small .json file to your Downloads folder. Store it somewhere safe — like a cloud folder or external drive.

How to import a backup

Open Campus Cash Pro in your browser, go to Settings, and click Import Data. Select your previously exported .json file. Your data will be restored immediately.

Tip: Export a backup once a month, or before switching computers or browsers. This only takes a few seconds.
Troubleshooting

Common questions

My data disappeared. What happened?
This usually happens when the file is opened in a private/incognito window, or if browser storage was cleared. Going forward, always use a normal browser window. If you have a backup .json file, you can restore it via Settings → Import Data.
The app looks broken or won't load properly.
Try a hard refresh: press Cmd + Shift + R on Mac or Ctrl + Shift + R on Windows. Make sure you're using a modern browser (Chrome 90+, Safari 14+, Firefox 88+, or Edge 90+).
Is Campus Cash Pro available on mobile?
Campus Cash Pro is designed for desktop and laptop use. Some features may open in a mobile browser, but the layout and experience are optimized for a computer screen. For the best results, use it on a desktop or laptop.
Can I use Campus Cash Pro offline?
Yes, after the first load it works fully offline. The only feature that requires an internet connection is loading the Google Fonts used for the app's typography — if you're offline, the app will fall back to a system font automatically.
Will my data sync between devices?
Not automatically. Use the export/import backup feature in Settings to move your data between devices. Export from your current device, then import on the new one.
Can I rename or move the HTML file?
Yes. The file name and location don't affect how the app works. Just keep it somewhere you can find it easily.
